JSON 說明：輕鬆看懂 n8n 的節點資料

JSON（JavaScript Object Notation）是 n8n 節點之間傳遞資料的格式，每個節點執行後輸出的資料都是 JSON，可用 Schema、Table、JSON 三種方式檢視。

什麼是 n8n 的 JSON？
{⚙}

n8n 節點執行後，輸出的資料以 JSON 格式儲存在 items[] 陣列中，每筆資料包在 { "json": {...} } 結構裡傳給下一個節點。

三種檢視方式
  • Schema — 欄位名稱與型別目錄
  • Table — 試算表格式，多筆比較
  • JSON — 完整原始結構，寫 expression 必看
  • 用 {{ $json["欄位"] }} 引用資料

JSON 語法規則（重點）
  • ● 鍵（Key）必須使用雙引號 " " 包起來。
  • ● 字串值也必須使用雙引號 " "。
  • ● 值可以是：字串、數字、布林值、陣列、物件或 null。
  • ● 鍵值對之間使用逗號分隔。
  • ● 最後一個鍵值對後面不能有逗號。
  • ● n8n 資料是 items[]陣列，每筆資料有"json": {}欄位。

常用引用語法範例：
{{ $json["name"] }} ← 取得字串欄位
{{ $json["id"] }} ← 取得數字欄位
{{ $json["address"]["city"] }} ← 巢狀物件
{{ $node["HTTP Request"].json["id"] }} ← 跨節點引用
